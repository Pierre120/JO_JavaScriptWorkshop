JavaScript Data Types

JavaScript is a dynamcially typed programming language. Thus, redefining a variable with a different data type is possible.

Sample Code:
let dynamicVar = 'Hi!';
dynamicVar = 629; // valid

Primitive Data Types

Number

Numbers can be written with or without decimals.

Sample Code:
let withDecimal = 34.12; // with decimal
let withoutDecimal = 12; // without decimal

String

A string (or a text string) is a series of characters enclosed with either a single quote (') or a double quote ("). Backticks (`) can also be used.

let single = 'Single quote'; // string enclosed with 2 single quotes
let double = "Double quote"; // string enclosed with 2 double quotes
let mix1 = 'There is a pair of "double quoted string" inside this single quoted string';
let mix1Alternate = 'There is a pair of \'double quoted string\' inside this single quoted string'; // using escape character
let mix2 = "There is a pair of 'double quoted string' inside this double quoted string";
let mix2Alternate = "There is a pair of \"double quoted string\" inside this double quoted string"; // using escape character
let backticked = `I got a ${single} string`; // getting and adding the string value of `single`

Boolean

Booleans can only have two values: true or false.

Sample Code:
let trueBool = true;
let falseBool = false; // valid
